# WellPay

WellPay is a health financing platform designed for informal workers in Nigeria, providing access to micro-insurance and health credit.

## Problem

Millions of informal workers lack access to affordable healthcare financing. Most rely on out-of-pocket payments, leading to delayed treatment or financial hardship.

## Solution

WellPay combines:

- Health savings wallet
- Micro-insurance
- Instant health credit

This ensures users can access care when needed without financial strain.

## Features

- Health Wallet (save for medical expenses)
- Insurance Plans (affordable monthly coverage)
- Health Credit (instant emergency loans)
- Clinic Finder
- Admin dashboard for managing loans and plans

## How It Works

1. Users save small amounts regularly
2. Build eligibility for credit
3. Activate insurance over time
4. Access clinics when needed

## Tech Stack

- Adalo (No-code app builder)
- Firebase (optional for backend services)
- USSD integration (planned)
